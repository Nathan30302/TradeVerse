Gmail App Password (for bulk send here)

Logging into Gmail in your phone or browser uses your normal password. TradeVerse sends mail from the server (like Outlook “SMTP”). Google does not allow your normal password there, so you create a separate App Password—one long code—to paste into MAIL_PASSWORD.

  1. Open Google Account → Security.
  2. Turn on 2-Step Verification if it is not on yet.
  3. Search for App passwords (or “App passwords” under 2-Step Verification).
  4. Create one for “Mail” / “Other” → copy the 16-character password (spaces optional).
  5. On your host (e.g. Render), set MAIL_USERNAME = your Gmail address (e.g. {{ support_email }}), MAIL_PASSWORD = that App Password, redeploy, then use Send test on this page.

If you prefer not to set this up, use the panel on the right: copy addresses and send one campaign from Gmail yourself (use BCC).
