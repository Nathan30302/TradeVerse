================================================================================
                           TROUBLESHOOTING
================================================================================

Build fails?
  → Check "Build Log" in Render dashboard
  → Look for Python errors
  → Verify requirements.txt is correct

App won't start?
  → Check "Deploy Log" for errors
  → Verify DATABASE_URL environment variable is set
  → Check Render logs for Flask errors

Can't access database?
  → Make sure PostgreSQL instance shows "Available"
  → Verify DATABASE_URL is copied correctly
  → Check connection string format

Slow to load?
  → App might be waking from sleep (30s is normal)
  → This only happens after 15 min inactivity
